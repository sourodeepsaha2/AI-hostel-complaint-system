# рҹҸ  HostelCare - AI Powered Hostel Complaint Management System

## вҡҷпёҸ Installation

### Clone Repository

```bash
git clone https://github.com/sourodeepsaha2/AI-hostel-complaint-system.git
cd AI-hostel-complaint-system
```

---

## Frontend Setup

```bash
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## Backend Setup

Navigate to backend:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Create `.env`

```env
PORT=8000
MONGO_URI=your_mongodb_connection_string
```

Run backend:

```bash
npm run dev
```

Backend runs on:

```bash
http://localhost:8000
```
